A web browser is what most people use to access different content on the internet. Whether it is browsing websites, logging into email, playing some games in a browser or using a web enabled application the browser is the primary application. Internet security is very important.

Use the web safely

  • Keep your software (operating system, anti virus and browser) up to date:
      • You should always run updates to keep your operating system current and be running the latest version of your web browser and virus definitions to help protect yourself from vulnerabilities.

  • Be careful what you download:
      • Only download items from trusted sources/sites and make sure to scan them for viruses after download before running.

  • Go to secure websites:
      • Connect to websites that use security https in their URL in order to ensure your data sent between you and the site is encrypted and protected.
